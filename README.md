# Tarea-5-Visualizacion-de-Datos-y-Tendencias-en-Big-Data---Paula-Pinzon

Este proyecto consiste en el análisis de un conjunto de datos epidemiológicos sobre el hantavirus mediante el uso de herramientas de Big Data y visualización de datos, específicamente Power BI.
El objetivo principal es transformar grandes volúmenes de información en representaciones visuales claras que permitan identificar patrones, tendencias y relaciones relevantes en los datos. Para ello, se realizó un proceso de modelado de datos, creación de relaciones entre múltiples tablas y diseño de un dashboard interactivo.
El dataset utilizado integra información global sobre el hantavirus, incluyendo:

Casos confirmados
Muertes
Hospitalizaciones
Distribución geográfica por países y regiones
Tendencias temporales
Características clínicas de los pacientes

A partir de este análisis, se desarrollaron diferentes visualizaciones (gráficos de barras, líneas, tablas, indicadores KPI y gráficos circulares) que permiten explorar el comportamiento epidemiológico del virus de forma intuitiva.
Este proyecto evidencia el valor del uso de herramientas de inteligencia de negocio para convertir datos en información útil, facilitando la comprensión de fenómenos complejos en el ámbito de la salud pública.
